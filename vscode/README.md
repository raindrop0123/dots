# Files

---

+ settings.json: settings offered by VS Code or its extensions
+ keybindings.json: keybindings
+ File Path:
  + Linux: `~/.config/Code/User/`
  + MacOS: `~/Application Support/Code/User`
  + Windows: `%AppData%\Code\User`
