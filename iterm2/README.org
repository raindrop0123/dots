* How to restore iterm2 settings?

1. =defaults delete com.googlecode.iterm2=
2. =rm ~/Library/Preferences/com.googlecode.iterm2.plist=
3. =rm -rf ~/.config/iterm2=

* How to setup the iterm2 settings on other mac?

1. Go to =Preference= -> =General= -> =Preferences= -> =Load preferences from a custom folder or URL=
2. Paste =https://raw.githubusercontent.com/raindrop0123/dots/main/iterm2/com.googlecode.iterm2.plist=
